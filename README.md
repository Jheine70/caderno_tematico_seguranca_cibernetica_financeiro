



📌 Sobre



Projeto acadêmico que analisa segurança cibernética e autenticação biométrica na proteção de contas bancárias em um cenário de crescimento das transações digitais.



🎯 Pergunta Norteadora


Como a biometria pode fortalecer a segurança das contas bancárias no ambiente financeiro digital?


🧠 Conteúdos do Projeto


- Segurança digital no sistema financeiro
- Proteção de contas bancárias
- Uso da biometria na autenticação
- Riscos de fraudes cibernéticas
